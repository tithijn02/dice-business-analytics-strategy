# 🎟️ DICE Strategy Case Study: Making Every Seat Count

A business analytics consulting project solving demand uncertainty, no-shows, and personalization challenges in event ticketing using data-driven solutions.

## 🚨 The Problem

DICE faces three key challenges limiting growth and efficiency:

- **Unpredictable demand** → ~40% tickets sold last-minute  
- **High no-shows** → Up to 30% empty seats despite sold-out events  
- **Weak personalization** → Low engagement and discovery inefficiencies  

👉 Result: Lost revenue, poor user experience, and reduced partner trust

## 💡 The Solution

This project proposes three data-driven solutions:

1. **Demand Forecasting Model**
   - Predict ticket sales using time-series + behavioral data  

2. **No-Show Prediction Engine**
   - Identify at-risk attendees and optimize ticket allocation  

3. **Hybrid Recommendation System**
   - Personalize event discovery using collaborative + content-based filtering

## 📊 Expected Impact

- 📈 Improved demand planning and marketing efficiency  
- 🎟️ Higher attendance and revenue optimization  
- 🔁 Increased user engagement and retention  
- 🤝 Stronger promoter trust and platform credibility

## 🛠️ Approach

- Data-driven problem structuring  
- Predictive modeling (forecasting + classification)  
- Recommendation systems  
- Strategic frameworks (Design Thinking, Agile, Product Management)
